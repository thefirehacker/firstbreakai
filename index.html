What is First Break AI?

First Break AI is a free, community-driven cohort for anyone who wants their first break in AI. The cohort is completely agentic — AI agents are woven into every layer of the learning experience. An agentic MCP server answers your questions, AI agents validate your code against rubrics, track your progress across devices, and guide you through a six-step roadmap from your first commit to a capstone project.

Unlike traditional online courses that give you pre-recorded videos and quizzes, First Break AI uses real AI agents to close the feedback loop instantly. You write code, the AI agent checks it, tells you what passed, and suggests your next step. This is what makes agentic AI different from a chatbot — the AI agents take real actions, not just answer questions.

Who is it for?

Students, professionals, career switchers, or anyone curious about AI. No CS degree required — you need comfort reading code and running terminal commands. No applications, no prerequisites. The cohort runs May–July 2026 with weekly Friday office hours, but all materials stay online permanently.

How AI agents power the cohort

  • AI agents answer questions in context — the on-site assistant and CLI agent know your syllabus and your current step
  • AI agents validate your work — run firstbreakai validate and an AI agent checks your code against the rubric
  • AI agents track your progress — log in with Discord and your progress syncs across CLI, website, and MCP server
  • AI agents integrate with your IDE — install the MCP server in Cursor, Claude Desktop, Claude Code, or OpenAI Codex

This agentic infrastructure is built on FetchLens.ai — the same platform that powers AI agent observability for websites. Install the npm package @aiedx/firstbreakai to get a CLI and local MCP server in one command.
